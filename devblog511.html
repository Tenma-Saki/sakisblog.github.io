测试船改动 - 封测13.1

  • 战舰世界 开发者日志 平衡性调整

根据测试结果，我们对英联邦巡洋舰线、维多利亚、威斯巴登、朱庇特42以及筑摩II作出了平衡性调整。


杰克逊港，泛英II级巡洋舰 削弱

  • 主炮装填时间增加：9.1s → 9.6s

卡拉多克，泛英III级巡洋舰 削弱

  • 主炮装填时间增加：8.0s → 8.5s

红堡，泛英V级巡洋舰 削弱

  • 主炮装填时间增加：6.0s → 6.6s
  • 反潜机空袭距离缩短：8km → 7km

    • 反潜机其他参数已按如下公布的方式修改 官网链接

  • 添加了与霍巴特类似的潜艇监视设备消耗品
  • 移除了船体的鱼雷防护

红堡号是英联邦V级巡洋舰德里的克隆战舰

霍巴特，泛英VI级巡洋舰 削弱

  • 主炮装填时间增加：6.5s → 7.1s
  • 反潜机空袭距离缩短：8km → 7km

乌干达，泛英VII级巡洋舰 削弱

  • 主炮装填时间增加：6.5s → 7.1s
  • 反潜机空袭距离缩短：9km → 8km

    • 反潜机其他参数已按如下公布的方式修改 官网链接

  • 添加了与奥克兰类似的潜艇监视设备消耗品
  • 移除了船体的鱼雷防护

奥克兰，泛英VIII级巡洋舰 削弱

  • 主炮装填时间增加：10s → 11s
  • 水面隐蔽增大：12.3km → 13.28km

    • 其他隐蔽相应增加

遭遇，泛英IX级巡洋舰 削弱

  • 主炮装填时间增加：8.9s → 9.8s
  • 水面隐蔽增大：12.3km → 12.96km

    • 其他隐蔽相应增加

朱庇特42，英国VII级驱逐舰 接近无变化

  • 主炮装填时间增加：4.5s → 6.5s
  • AP参数已更改：

    • 标伤增加：1700 → 2500

    引信触发阈值减小：20mm → 12mm

维多利亚，西班牙IX级战列舰 增强

  • 改进了主炮的齐射角
  • 主炮组Sigma增加：1.8 → 2
  • 交换了常规炮弹与替代炮弹(F)。高速轻弹现在是常规炮弹，低速重弹是替代炮弹。
  • 替代炮弹AP参数已更改：

    • 标伤增加：13800 → 14200

  • 替代炮弹HE参数已更改：

    • 标伤增加：5250 → 5850

    点火率增加：30% → 40%

威斯巴登，德国VIII级巡洋舰 削弱

  • 主炮装填时间增加：5s → 6s
  • 对海搜索消耗品参数已更改：

    • 水面探测范围缩小：6km → 5km

    鱼雷探测范围缩小：4km → 3.5km

    生效时间缩短：120s → 100s

  • 烟雾消耗品参数已更改：

    • 消耗品生效时间减少：20s → 15s

    冷却时间增加：160s → 180s

筑摩II，日本IX级巡洋舰 接近无变化

  • 因修复模型错误，减小1°舰载鱼雷射角

请注意，开发博客中的所有信息都是初步信息。在测试过程中，公布的调整和功能可能会多次更改。最终信息将公布在我们的游戏网站上。
